 #  20230608图森私有化部署详情



## Release Notes

- 网关运维界面增加“车辆名称”字段，且搜索栏支持通过名称或者VIN码搜索网关设备；
- 网关日志界面，左侧车辆列表支持按照名称或者VIN码方式进行显示；
